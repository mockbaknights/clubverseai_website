# Club Type Images

Add your club type images here with these exact filenames:

- `tennis.jpg` - Tennis club image
- `golf.jpg` - Golf club image
- `pickleball.jpg` - Pickleball image
- `padel.jpg` - Padel club image
- `curling.jpg` - Curling club image
- `multi-sport.jpg` - Multi-sport club image

**Image recommendations:**
- Format: JPG or PNG
- Size: 1200x800px or larger (will be automatically optimized)
- Aspect ratio: 3:2 or 16:9 works best

The images will be automatically optimized by Next.js Image component.
